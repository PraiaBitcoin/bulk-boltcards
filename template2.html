ready-only wallet

This QR code allows the user to import the LNBits Lightning Wallet in Read-Only mode.

The user can create invoices with that, and see the transactions, but can’t move the funds.
LNDHub Invoice URL

how to import on bluewallet

After install and open Bluewallet, click on “Add Wallet”, “Import Wallet”, press on “Read code or import file”, scan the QR code of the “LndHub Admin” or "LndHub Invoice" and wait a few seconds to the wallet be imported.
